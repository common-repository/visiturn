Add Visiturn code in wordpress all pages to get site stats.

== Description ==

Adding visitrun code in wordpress all pages before < / body > tag and get your website stats in Visiturn network.

== Installation ==

1. Upload the `visiturn` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit *Appearance > Visiturn*
4. Place the code which given by Visiturn website and save it
5. Visit your account on Visiturn website and observe your website stats.

== Frequently Asked Questions ==

= How can I add Visiturn code in my theme? =
You can Install the wordpress plugin "Visiturn" and go to the "Visiturn" menu on "Settings" section. Add the code which you get from us and click "Save Changes". Thats all !!. You can check your website stats in Visiturn network.
